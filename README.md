## Features

* Music in background
* Changing background images
* Progress Bar
* Footer text
* [6-23/7](https://ks-code.mintlify.app/wstep) Support

## How To Use

1. Download repo and put to your `resources` folder.
2. Add `start ks_loading` to your `server.cfg`.
3. Start server.
